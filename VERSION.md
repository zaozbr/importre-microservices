# Versionamento

Este projeto segue versionamento semantico (SemVer) adaptado:

- **Patch** (terceiro numero): correcoes de bug, ajustes pequenos.
- **Minor** (segundo numero): novas funcionalidades, plugins, melhorias.
- **Major** (primeiro numero): mudancas que quebram compatibilidade.

## Versao atual

v1.0.0

## Historico

- v1.0.0 — Primeira versao versionada. Arquitetura de plugins de fontes, dashboard ajustado, STOP/RESTART robusto, reprocessamento de falhas corrigido.
